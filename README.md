## 🚀 Características del Proyecto

Este portafolio ha sido diseñado y desarrollado desde cero centrándose en el rendimiento, la estética "tech/dark" y la experiencia de usuario:

- **Diseño Dark UI & Glassmorphism:** Interfaz moderna con tonos oscuros, detalles en rojo (`#ff0000`) y barras de navegación translúcidas.
- **Efecto CRT/Scanlines:** Un toque retro-tecnológico sutil implementado con CSS puro.
- **Bento Grid Layout:** Sistema de cuadrículas moderno para mostrar la trayectoria profesional.
- **Animaciones Nativas:** Efectos de máquina de escribir (Typewriter), scroll revelador (Intersection Observer) y carrusel infinito (Marquee) sin librerías externas.
- **Totalmente Responsivo:** Adaptado para funcionar perfectamente en dispositivos móviles, tablets y monitores ultrawide.
- **Sistema de Modales Dinámico:** Visualización detallada de proyectos inyectada mediante JavaScript.

## 🛠️ Stack Tecnológico

El proyecto fue construido utilizando tecnologías web nativas para garantizar la máxima velocidad de carga y cero dependencias innecesarias:

- **HTML5:** Estructura semántica y accesible.
- **CSS3:** Variables nativas, Flexbox, CSS Grid, animaciones `@keyframes` y Media Queries.
- **Vanilla JavaScript:** Manipulación del DOM, Intersection Observer API y lógica de interfaz.

*(Mi stack general de desarrollo incluye además: Python, React.js, Node.js, SQL/NoSQL, Git/GitHub, entre otros).*

## ⚙️ Instalación y Uso Local

Si deseas clonar este repositorio para ver el código en tu máquina local:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/OsvaLogic/Portafolio-Osvaldo-Sanhueza.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd Portafolio-Osvaldo-Sanhueza
   ```
3. Abre el archivo `index.html` en tu navegador de preferencia. No requiere configuración de servidores ni instalación de paquetes (`npm`).
